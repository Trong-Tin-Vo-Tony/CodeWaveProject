Position Description (Cyber Security Roles)

Below are the roles we are hiring for. Each position includes title, employment type, reporting line, summary, duties and requirements.

Cyber Security Analyst

Mode of Employment: Full-time
Reports to: Security Operations Center (SOC) Manager

Position Summary: Monitor and analyze security events to detect and prevent threats.

Duties & Responsibilities

  • Monitor network traffic, logs and alerts for suspicious activity.
  • Investigate and respond to potential incidents.
  • Apply security patches and updates.
  • Support compliance and security reporting.

Requirements

  • CISSP or equivalent (or working toward).
  • Strong analytical/problem-solving and communication skills.
  • Familiar with SIEM, IDS/IPS and endpoint protection.

Network Security Engineer

Mode of Employment: Full-time
Reports to: IT Infrastructure Manager

Position Summary: Design and maintain secure network systems, firewalls and access controls.

Duties & Responsibilities

  • Configure/maintain firewalls, routers, VPNs.
  • Implement IDS/IPS and network access controls.
  • Run security audits and hardening.
  • Troubleshoot security-related network issues.

Requirements

  • CCNA Security / CompTIA Security+ or similar.
  • Knowledge of TCP/IP, VPNs, routing, SOAR/automation.
  • Strong documentation and stakeholder communication.

Security Consultant

Mode of Employment: Full-time or Contract
Reports to: Head of Cybersecurity Consulting

Position Summary: Advise clients on security best practices, risk and compliance.

Duties & Responsibilities

  • Perform risk/vulnerability assessments.
  • Develop security strategies and roadmaps.
  • Guide on ISO 27001, SOC 2, GDPR and related standards.
  • Deliver workshops and stakeholder training.

Requirements

  • CISM / ISO 27001 Lead Auditor or equivalent.
  • Client-facing communication and presentation skills.
  • Proven framework design/implementation experience.

Penetration Tester

Mode of Employment: Full-time or Project-based
Reports to: Offensive Security Team Lead

Position Summary: Simulate cyberattacks to identify vulnerabilities and recommend fixes.

Duties & Responsibilities

  • Conduct app, network and cloud pentests.
  • Document findings and remediation steps.
  • Use tools like Burp Suite, Metasploit, Kali.
  • Participate in red team exercises.

Requirements

  • GPEN / OSCP / CEH or similar.
  • Strong exploit and scripting knowledge.
  • Clear, concise reporting skills.

Incident Response Specialist

Mode of Employment: Full-time
Reports to: Incident Response Manager

Position Summary: Lead investigation, containment and recovery during cybersecurity incidents.

Duties & Responsibilities

  • Coordinate incident triage, containment and eradication.
  • Perform forensics on affected endpoints/servers.
  • Create and maintain IR playbooks and lessons learned.
  • Interface with legal, compliance and (if needed) law enforcement.

Requirements

  • GCIH / CHFI or equivalent.
  • Malware analysis and digital forensics skills.
  • Ability to operate under pressure and on-call.

Preferred Qualifications

  • Certifications: CISSP, GPEN, or cloud security certifications (AWS Certified Security - Specialty).
  • Experience with SOAR tools.
  • Prior experience in regulated industries (finance, healthcare, defense).

Soft Skills & Competencies

  • Strong analytical and problem-solving skills.
  • Excellent communication skills (technical & non-technical audiences).
  • Proactive and continuous improvement mindset.
  • Teamwork and independent work capability.
